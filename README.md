# Seashore Electric Website

This is a static website package ready for GitHub Pages, Cloudflare Pages, or another static host.

## Files
- `index.html` — main webpage
- `style.css` — design and mobile layout
- `script.js` — mobile menu and automatic copyright year
- `favicon.png` — browser tab icon
- `images/` — company logo and real project photos

## Publish with GitHub Pages
1. Create a free GitHub account.
2. Create a new public repository named `seashore-electric-website`.
3. Upload all files and the `images` folder from this package.
4. Open the repository's **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/ (root)`, then save.
7. Once published, enter `www.seashoreva.com` under **Custom domain**.
8. Add the DNS records GitHub provides in your domain manager.
9. Turn on **Enforce HTTPS** after DNS finishes updating.
